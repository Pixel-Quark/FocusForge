# 🌸 FocusForge

A cozy, gamified productivity app that turns tasks into a fun and emotional experience.

## ✨ Features

* 🎯 Task management with priorities
* ⚡ XP, levels, and streak system
* 🧸 Animated mascot with emotions
* 💬 Interactive speech bubble feedback
* 🎨 Soft pastel UI design
* ⌨️ Keyboard shortcuts for speed

## 🚀 Tech Stack

* HTML
* CSS (custom animations)
* Vanilla JavaScript

## 💡 Future Ideas

* Sound effects
* Theme switcher
* AI personality system
* Mobile version
